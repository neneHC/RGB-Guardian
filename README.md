🛡️ #RGB Guardian

RGB Guardian é um jogo de defesa rítmica e reflexos rápidos desenvolvido inteiramente em HTML5, CSS3 e JavaScript puro. O objetivo é proteger a base contra blocos coloridos que descem em velocidade progressiva, utilizando disparos de cores correspondentes.
🎮 ##O Jogo

No controle do sistema de defesa, o jogador deve alternar entre as cores Red (Vermelho), Green (Verde) e Blue (Azul) para destruir os inimigos antes que eles atinjam o limite inferior da tela.
Principais Características:

    Dificuldade Progressiva: A velocidade dos inimigos aumenta gradualmente conforme você sobrevive.

    Feedback Visual: Sistema de partículas para explosões e um efeito de flash na tela em caso de Game Over.

    Multiplataforma: Totalmente jogável tanto no desktop (teclado) quanto em dispositivos móveis (touch).

🕹️ ##Como Jogar

O jogo utiliza uma mecânica de correspondência de cores. Um tiro só destrói um bloco se ambos forem da mesma cor.
Controles (PC):

    Tecla A: Dispara raio Vermelho.

    Tecla S: Dispara raio Verde.

    Tecla D: Dispara raio Azul.

###Controles (Mobile):

    Toque nos botões coloridos na parte inferior da tela para disparar.

🛠️ ##Tecnologias Utilizadas

Este projeto demonstra a aplicação de conceitos de desenvolvimento front-end e lógica de jogos:

    DOM Manipulation: Gerenciamento de entidades (inimigos e projéteis) em tempo real.

    CSS Animations: Efeitos de explosão e interface responsiva.

    Game Loop: Implementado com requestAnimationFrame para alta performance e fluidez.
